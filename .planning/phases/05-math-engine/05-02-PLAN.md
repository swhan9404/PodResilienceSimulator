    **Create `src/optimizer/sweep.ts`** (per D-07, D-08, D-09, D-10):

    Export one function: `computeSweep(input: OptimizerInput, rangeOverride?: { podRange?: [number, number]; workerRange?: [number, number] }): SweepPoint[]`

    **Auto-range derivation** (per D-07):
    1. Compute weighted average latency: `avgLatencyMs = sum(profile.latencyMs * profile.ratio for each profile)`
    2. Compute service rate per worker: `mu = 1000 / avgLatencyMs` (convert ms to per-second)
    3. Compute minimum total workers for stability: `minWorkers = Math.ceil(input.rps / mu)`
    4. Lower bound: `Math.max(1, Math.floor(minWorkers * 0.5))` total workers
    5. Upper bound: `Math.ceil(minWorkers * 2.5)` total workers
    6. Derive pod range: `podMin = 1`, `podMax = Math.min(Math.ceil(upperBound / 1), upperBound)` -- but clamp so podMax * workerMax <= 500 (D-08)
    7. Derive worker-per-pod range: `workerMin = 1`, `workerMax` = clamp so total grid <= 500

    **Practical range clamping for <500 grid points** (per D-08):
    - `podRange = rangeOverride?.podRange ?? [podMin, podMax]`
    - `workerRange = rangeOverride?.workerRange ?? [workerMin, workerMax]`
    - Compute gridSize = (podMax - podMin + 1) * (workerMax - workerMin + 1)
    - If gridSize > 500: reduce podMax and workerMax proportionally until gridSize <= 500
    - Minimum: both ranges must have at least 1 value

    **Sweep execution**:
    - For each podCount in [podMin..podMax] step 1 (D-08):
      - For each workersPerPod in [workerMin..workerMax] step 1 (D-08):
        - Compute `c_eff = computeEffectiveWorkers(workersPerPod, input.probes)` per pod
        - Total effective workers: `c = c_eff * podCount`
        - Total system capacity: `K = (workersPerPod + input.maxBacklogPerPod) * podCount` (per D-09: maxBacklog is fixed input)
        - Call `computeMMcK(input.rps, mu, c, K)`
        - Push `{ podCount, workersPerPod, totalWorkers: podCount * workersPerPod, result }` to array

    - Sort result array by `totalWorkers` ascending (stable sort, secondary sort by podCount ascending for same totalWorkers)
    - Return sorted array

    **Step 1: Create `src/optimizer/kneedle.ts`** (per D-04, D-05, D-06, D-10):

    Export one function:
    ```typescript
    export function findKneePoint(
      xs: number[],           // total workers (ascending) per D-04
      ys: number[],           // P_block (descending) per D-04
      sensitivity?: number,   // Kneedle sensitivity, default 1.0
      threshold?: number,     // Fallback P_block threshold, default 0.01 per D-06
    ): KneeResult
    ```

    **Kneedle algorithm** (per Satopaa et al. 2011, simplified for offline decreasing curve):

    1. **Guard**: if `xs.length < 3`, return `{ kneeIndex: null, fallbackIndex: computeFallback(ys, threshold), method: 'threshold' }`

    2. **Normalize to [0, 1]**:
       - `xMin = xs[0]`, `xMax = xs[xs.length - 1]`
       - `yMin = Math.min(...ys)`, `yMax = Math.max(...ys)`
       - `xRange = xMax - xMin || 1` (avoid division by zero)
       - `yRange = yMax - yMin || 1`
       - `xn[i] = (xs[i] - xMin) / xRange`
       - `yn[i] = (ys[i] - yMin) / yRange`

    3. **Compute difference curve for decreasing data**:
       - For decreasing curve, flip y: `diff[i] = (1 - yn[i]) - xn[i]`

    4. **Find local maximum of difference curve**:
       - `thresholdValue = (sensitivity ?? 1.0) / xs.length`
       - Scan i=1..length-2: if `diff[i] >= diff[i-1] && diff[i] >= diff[i+1]` and `diff[i] > thresholdValue`
       - Track the index with the highest diff value as `bestIdx`

    5. **Compute fallback** (per D-05, D-06):
       - Scan ys from index 0: find first index where `ys[i] < (threshold ?? 0.01)`
       - If none found, `fallbackIndex = null`

    6. **Return**:
       - If `bestIdx >= 0`: `{ kneeIndex: bestIdx, fallbackIndex, method: 'kneedle' }`
       - Else: `{ kneeIndex: null, fallbackIndex, method: 'threshold' }`
